# client

Инструкция:

Склонировать репозиторий, в корнейвой папке выполнить команду:
==============================================================

npm install

После установки всех зависимостей, стартовать приложение:
=========================================================

npm start

После компиляции, стандартно открывается localhost:3000 где и крутится приложение:
